# LOT Radio Tracks

Place your audio tracks in this folder to make them available for LOT Radio.

## Supported Formats
- `.mp3` - MP3 audio
- `.wav` - WAV audio
- `.ogg` - Ogg Vorbis audio
- `.m4a` - M4A audio

## Usage
1. Add audio files to this directory
2. The radio will automatically detect and play random tracks
3. When a track finishes, another random track will play

## Track Naming
- Use descriptive filenames (e.g., `ambient-drone-1.mp3`)
- Avoid special characters in filenames
- The filename (without extension) will be displayed to users
